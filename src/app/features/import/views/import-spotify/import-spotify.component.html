Von Spotify importieren

Importiere Playlisten über Spotify-Links.

Über diesen Dienst können keine Lieder von Spotify heruntergeladen werden. Es werden lediglich die Playlisten gespiegelt. Das bedeutet, dass alle Lieder auf dieser Plattform gesucht werden und zur passenden Playlist hinzugefügt werden.

Die Warteschlange ist leer.
