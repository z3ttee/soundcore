Spotify Import
Importiere existierende Playlisten von Spotify

Wie funktioniert der Import?

Du kannst Spotify Playlisten importieren, in dem du den Link zur Playlist angibst. Das System öffnet diese Playlist und sammelt alle darin enthaltenen Lieder. Diese Sammlung wird danach mit der Soundcore-Datenbank abgeglichen. Dabei werden Titel, Künstler und das Album des Lieds berücksichtigt. Bei Treffern, wird das Lied aus der Soundcore-Datenbank in eine neue Playlist hinzugefügt. Der Name dieser neuen Playlist trägt den gleichen Name wie die Spotify Playlist.
